My own distribution system is similar to Kafka: I was inspired to build such a complex project as a goal for myself to become a better programmer. Kafka is a distribution system written in Java by LinkedIn and used also by Netflix for its level of speed. Kafka uses brokers to send information; however, the information is sent in bytes, which may be the main reason for Kafka's record-breaking speed. I stop at each step to understand how Kafka is built and what reasons mega corporations use it.
